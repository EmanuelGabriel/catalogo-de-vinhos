Sistsema Catálogo de Vinhos


Este é um pequeno projeto que demonstra o uso de uma aplicação desenvolvida em Spring MVC e Spring Boot.

Foram utilizadas as seguintes ferramentas:

- Thymeleaf
- Swagger UI
- Spring Boot
- Spring Data
- Spring Security
- Maven
- PostgreSQL
- Heroku(Deploy)
- Spring Tool Suite(IDE)
